# Beaverforge IRL Popularization Plan (Zero Budget)

Owner identity for all public messaging: **trenn1x**

## Objective

Turn Beaverforge multiplayer into a local and online micro-phenomenon with no paid spend.

## Non-Negotiables

- Every post/profile/signature uses `trenn1x`.
- Every outreach message contains a playable link and one direct call-to-action.
- Keep sessions short and social: “play one match now.”

## Daily Operating Loop (45-60 min)

1. Launch a fresh share session:
   - `cd /Users/thomasverdier/arcane-duel`
   - `./scripts/start_trenn1x_session.sh 8093`
2. Host 2-4 live matches and record short clips/screenshots.
3. Post once in each free channel:
   - 1 Discord server
   - 1 Reddit thread
   - 1 short-form video platform post
4. Follow up with every person who clicked or responded.
5. End with one “tonight’s room code” post.

## Core Message Templates

## Cold DM

`yo it's trenn1x - built a free beaver strategy card game with online versus. want to run 1 fast match right now? <link>`

## Free Distribution Channels

- Existing friend groups + group chats
- Local college/high-school gaming Discords
- Reddit game-dev and indie-gaming communities (follow each sub rules)
- X / TikTok / Instagram Reels short clips
- Hacker News Show HN (when stable and polished)

## IRL Tactics That Cost $0

- Use your phone as a live “station” and challenge people to one match.
- Run “winner stays” sessions at social events.
- Keep a reusable room code in your notes app for fast invites.
- Ask every player to invite one cousin/friend immediately.

## Retention Hooks

- Always send rematch code after each match.
- Highlight streaks (“3-0 beaver king”) in chat posts.
- Share result screenshot + copied result text from in-game **Share Result**.

## Weekly Targets

- 50 unique players
- 100 multiplayer matches
- 20 players who return for a second session

## What To Track Manually

- Number of rooms created
- Number of matches started
- Number of players who came from invites
- Top channels generating joins

Use a simple markdown log in this repo if needed.
